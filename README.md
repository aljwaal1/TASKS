# مهامي الملوّنة

تطبيق Flutter عربي خفيف لإدارة المهام حسب الحالة:

- مطلوب: أحمر.
- تحت الإنجاز: برتقالي.
- منجز: أخضر.

## نظام التذكير (محدّث)

يدعم التطبيق الآن تذكيرين منفصلين بصوتين مختلفين:

1. **تنبيه الاقتراب** 🟡 — قبل موعد المهمة بالمدة التي تحددها (10/15/30/60/120 دقيقة، أو بدون)، بصوت خفيف مميز.
2. **تنبيه حلول الوقت** 🔔 — في اللحظة المحددة بالضبط، بصوت أوضح، ويعرض بيان المهمة كاملاً (العنوان + الحالة + الملاحظة).

يمكن اختبار الصوتين من شاشة المهام عبر أيقونتي الجرس بجانب زر الترتيب.

## اختبار التذكير

1. افتح التطبيق.
2. أضف مهمة جديدة.
3. اختر تاريخ ووقت التذكير بعد دقائق قليلة، وحدد مدة التنبيه المبكر.
4. وافق على صلاحية الإشعارات إذا ظهرت.
5. اخرج من التطبيق وانتظر وقت التذكير.

## البناء

شغل GitHub Actions باسم:

`Build APK Direct`

بعد نجاح البناء سيظهر APK مباشرة داخل:

`apk/task-status-reminder.apk`

لا يوجد Release ولا AAB.

ملاحظة: ملفات الصوت المخصصة (`sound_resources/task_due.mp3` و `sound_resources/task_approaching.mp3`) تُنسخ تلقائيًا إلى `android/app/src/main/res/raw/` أثناء البناء عبر GitHub Actions.


## إصلاح الإشعارات

في هذه النسخة تم نقل طلب صلاحية الإشعارات بعد فتح واجهة التطبيق، وإضافة زر اختبار الإشعار الآن. كما تم استخدام جدولة أدق للتذكير مع fallback تلقائي إذا لم يسمح الجهاز بالجدولة الدقيقة.

طريقة الاختبار:
1. افتح التطبيق واضغط اختبار الإشعار الآن.
2. وافق على صلاحية الإشعارات إذا ظهرت.
3. أضف مهمة بتذكير بعد 3 دقائق.
4. اخرج من التطبيق وانتظر وقت التذكير.

إذا لم يظهر الإشعار على بعض الأجهزة القديمة، يجب تفعيل إشعارات التطبيق يدويًا من إعدادات الهاتف وإيقاف توفير البطارية لهذا التطبيق.
